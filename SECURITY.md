# Security Policy

## Supported Versions

This project is community-maintained. Security fixes are provided on a best-effort basis.

## Reporting a Vulnerability

Please report security issues **privately**.

- Preferred: open a GitHub Security Advisory in the repository
- Alternative: create a GitHub issue and mark it as sensitive (avoid posting exploitation steps)

When reporting, include:
- Salesforce edition (Dev/Enterprise/etc.) and API version
- Steps to reproduce
- Impact assessment (data exposure, privilege escalation, etc.)
- Any relevant logs / screenshots (redact secrets)
